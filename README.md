# Recipe Management System

A web app to manage recipes, search for recipes, submit reviews, and manage user profiles.

## Features
- Search for recipes
- Add and manage reviews
- User profile management
- Responsive design with React.js

## Tech Stack

- HTML, CSS, JavaScript
- (Optional) Backend API or local storage
